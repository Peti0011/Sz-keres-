Ez a projekt egy Python alapú szókereső algoritmust valósít meg, amely egy betűrácsban keres előre megadott szavakat több irányban, majd az eredményt vizuálisan is megjeleníti.
Szavak keresése egy 2D betűrácsban
8 irányú keresés:
vízszintes (balra, jobbra)
függőleges (fel, le)
átlós irányok
Találatok kiírása konzolra
Grafikus megjelenítés a matplotlib segítségével:
eredeti rács
színesen kiemelt találatok
